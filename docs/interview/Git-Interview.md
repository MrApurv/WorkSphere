# Git Interview Questions

## Q1 What is Git?

Git is a distributed version control system.

---

## Q2 Difference between Git and GitHub?

Git

Version control software.

GitHub

Cloud platform hosting Git repositories.

---

## Q3 Difference between git add and git commit?

git add

Stages changes.

git commit

Creates a snapshot.

---

## Q4 Difference between git push and git pull?

git push

Uploads local commits.

git pull

Downloads latest commits.

---

## Q5 What is a branch?

An independent line of development.

---

## Q6 Why use branches?

To develop features safely without affecting main.

---

## Q7 What is a merge?

Combining one branch into another.
